# Notebooks

Notebooks are numbered in the order they should be run. Each corresponds to one challenge task.

| Notebook | Task | Purpose |
|---|---|---|
| `01_data_exploration_enrichment.ipynb` | Task 1 | Load, validate schema, enrich dataset |
| `02_eda.ipynb` | Task 2 | Exploratory analysis, trends, gaps, correlations |
| `03_event_impact_modeling.ipynb` | Task 3 | Event-indicator association matrix, validation |
| `04_forecasting.ipynb` | Task 4 | Access & Usage forecasts, scenarios, uncertainty |

Run from the repo root so relative paths to `data/` resolve correctly, e.g.:

```bash
jupyter notebook notebooks/01_data_exploration_enrichment.ipynb
```
